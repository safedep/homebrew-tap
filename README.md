# SafeDep Homebrew Tap

Tap SafeDep homebrew repository

```bash
brew tap safedep/tap
```

Install [vet](https://github.com/safedep/vet) using `homebrew`

```bash
brew install safedep/tap/vet
```

Upgrade `vet`

```bash
brew update && brew upgrade safedep/tap/vet
```
